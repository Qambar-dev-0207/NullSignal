PROTOCOL: SEVERE BLEEDING

01

Direct Pressure

Apply firm, steady, direct pressure to the wound using a clean cloth or sterile bandage. Do not lift the dressing to check if bleeding has stopped.

02

Elevation

If possible, elevate the injured limb above the level of the heart while maintaining pressure. This reduces blood flow to the site of injury.

03

Pressure Points

If bleeding continues, apply pressure to the nearest major artery (brachial for arms, femoral for legs) between the wound and the heart.
